# Item Catalog
Udacity Project for the Full Stack Web Developer Nanodegree Program

# About
A data driven app that presents a list of items in 5 differents categories: 
	- Food
	- Animals
	- Sports
	- City
	- Fashion  

## stack 
	- Server: Python
	- FWK: Flask
	- RDBMS: postgresql
	- ORM: SqlAlchemy
