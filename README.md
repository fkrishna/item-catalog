# Item Catalog
Udacity Project for the Full Stack Web Developer Nanodegree Program

# About
A data driven app that presents a list of items in 5 different categories: 
	- Food<br>
	- Animals<br>
	- Sports<br>
	- City<br>
	- Fashion<br>  

## stack 
	- Server: Python
	- FWK: Flask
	- RDBMS: postgresql
	- ORM: SqlAlchemy
